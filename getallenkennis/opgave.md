# Zoned Decimal van een decimaal getal.

> Van decimaal naar binair:

845 / 2 = 422, rest: 1
422 / 2 = 211, rest: 0
211 / 2 = 105, rest: 1
105 / 2 =   52, rest: 1
  52 / 2 =   26, rest: 0
  26 / 2 =   13, rest: 0
  13 / 2 =     6, rest: 1
    6 / 2 =     3, rest: 0
    3 / 2 =     1, rest: 1
    1 / 2 =     0, rest: 1
